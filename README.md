# '패턴으로 익히고 설계로 완성하는 리액트' 스터디 📚

##### 스터디 기간 - 2025.04 ~ 진행중

## ✨ 목표

- 리액트의 안티 패턴을 파악하고 좋은 코드 작성 방법에 대해 학습한다.
- 리액트에서 사용 가능한 다양한 디자인 패턴을 학습하고 실제 코드에 적용해본다.
- 학습한 내용을 설명할 수 있도록 하여 자신의 지식으로 만든다.
- 함께 공부하며 서로에게 동기부여가 되어준다.

## 📂 Contents

1. 리액트 안티패턴 소개
2. 리액트 필수 개념 이해
3. 리액트 애플리케이션 구조
4. 리액트 컴포넌트 설계하기
5. 리액트 테스팅
6. 일반적인 리팩터링 기법 살펴보기
7. 리액트에서의 테스트 주도 개발
8. 리액트 데이터 관리
9. 리액트 설계 원칙 적용
10. 합성 패턴
11. 리액트 계층 구조 애플리케이션
12. E2E 프로젝트 구현하기
13. 리액트 안티패턴 원칙 돌아보기

## 📏 규칙

- 매주 토요일 10시 오프라인 모임
- 정해진 분량만큼 학습, 적용하고 정리한 내용 발표하기
- 진도에 맞는 질문 4개 이상 준비하여 스터디 시간에 질문과 답변하기
